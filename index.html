Scpswl

Un script pour sniffer les packets réseaux sur les deux environements Windows et Linux et pour décoder les en-têtes IP et ICMP

View on GitHub Download .zip Download .tar.gz

Bienvenue à la page de notre outil SCPSWL.

Nous vous présentons un script Médico-légal pour sniffer le trafic réseau d'un hôte cible ou d'un scanneur réseau qui vous permettera de sniffer et décoder les en-têtes IP et ICMP des packets réseau.

Avant de commencer

Notre scanner va utiliser une bibliothèque tierce appelée netaddr, ce qui vous permettra d'avoir un retour dans un masque de sous-réseau tels que 192 .68. ./24

Téléchargez de manière appropriée la bibliothèque à partir d'ici: http://code.google.com/p/netaddr/downloads/list

Comment l'utiliser

Ouvrez le fichier SCPSWL.py dans un éditeur de texte / code. Dans les variables suivantes , précisez l'adresse ou le sous-réseau que vous voulez sniffer:

L'hôte à écouter

host = "192.168.2.150"

Sous-réseau ciblle

subnet = "192.168.2.0/24"

Sur un terminal ou un prompt shell cmd.exe, exécutez le script comme suit:

syscore$ python scanner.py ou c:\Python27\python.exe scanner.py
